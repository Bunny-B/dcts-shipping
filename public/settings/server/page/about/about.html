The Mission

I loved using TeamSpeak back then and hosting my own server. It was such a cool feeling, and YATQA was such an amazing tool. It felt like having some sort of power. When i switched to Discord it was a big change, but also a great one!

Being free from the Slot Limit TeamSpeak had was such a relief, and Discord looked awesome too! My only problem with Discord was that you dont "really own" the server you have and that you have some sort of restrictions.

I've seen some alternatives like TeaSpeak (not TeamSpeak !) and talked with the creator back then. The project was cool but it had a few issues: Development was slow because only he worked on it, the controversy with TeamSpeak banning TeaSpeak Servers (TeamSpeak Client and TeaSpeak Server were compatible) and many people used the TeamSpeak Client as the TeaSpeak Client was very buggy (TOS Violation).

Many people also thought that TeaSpeak was simply a cracked TeamSpeak Server Version. So we still need a alternative...


But... isnt this a discord clone then?!

Well this depends on how you see it. I did have the goal to make it look similar to Discord so it wouldnt look so "different" for the user. The goal was not to copy Discord and never will be. The mission is to create a platform thats self hostable like TeamSpeak and looks a bit like Discord. In short: "A self-hosted Discord".
