What's new

      Role Update

      • Role Editor
        • Edit Roles
        • Create Roles
        • Delete Roles
        • Change Role Color
        • Change Role Name
        • Change Role Display Setting (show in member list)
        • Change Role Permissions
        • Change Role hierarchy (role sorting)

      Channel Update

      • Edit Channel
        • Its now possible to edit a channel by right clicking it and selecting "Edit Channel". There you can change the channel name and the permissions for roles.

      Server Update

      • SSL Support
        • SSL is required if you host your server on a maschine and access it from another device. Otherwise some features like the "Copy User ID" will not work as browsers like Chrome will block it on "unsecure sources". Also SSL will encrypt the Traffic which is another big plus. A few new settings have been added to the config file.

      General Changes

      • Member List Update
        • The Member list will only show members that are allowed to view the current channel.
      • Channel List Update
        • If a category doesnt have any channel a member/role is allowed to see, the entire category is hidden from the channel tree
        • If a category doesnt contain any channel, it is hidden from members that dont have the Manage Channels permissions
      • Group List Update
        • It is possible to hide groups from users now, tho i didnt fully implement this feature. This means while you can edit the config.json file to hide certain groups for certain roles, its not possible to do this using the web client/ui. This will be implemented in a feature update.
